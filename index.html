Botiga virtual

PHP

La boriga virtual ha sigut creada per practicar amb PHP utilitzant una base de dades local la cual registra totes les compres, productes.

Planes de la pagina

Unicament pot funcionar gracies a una base de dades registrada.

· Registre Seccio per registrar-se el usuari on es guarden les dades en la base de dades.

· Login Seccio per loguerar-se el usuari on es comparen les dades (amb la contrasenya encriptada amb md5) de la base de dades

· Botiga On es mostren tots els productes colocats en la base de dades, amb control de errors per quan no hi ha stock.

· Administrador Es la plana on unicament pot entrar el usuari administrador per Insertar, Eliminar, Modificar els productes.
